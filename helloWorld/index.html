Hello World!

Welcome to Javascript! Javascript runs natively in your browser (client) to make webpages interactive. Let's first get familiar with how Javascript is intimately connected with HTML and the browser, and then start using some variables, operators, and expressions. Follow the directions below:

  1. We want to see Javascript's output and error messages (for debugging), so let's use the Chrome/Firefox built-in Developer Console. To open the console, you can use a shortcut key combo: On OSX, use cmd+option+j. On Windows/Linux, use ctrl+shift+j. It should pop up on the botton or right hand side of your browser. Next to the > symbol, write console.log("Hello World!"); and press enter. Next, write 10+5; and press enter to perform the calculation.
  2. In Sublime, go an find an empty <script> element at the bottom of this HTML document's body. Inside of the <script> tags, write console.log("Hello World!"); and save the document. Refresh the page and check the console's output.
  3. Add another <script> element at the bottom of this page's body. Connect it to index.js by setting the element's src property equal to the relative path of index.js. Hint: it's in the same directory!
  4. Open index.js in Sublime and write console.log("Hello Again!"); on the first line. Save it, refresh index.html, and check the console.
  5. Nice work! Let's head back to the slides to learn about Variables, Operators, and Expressions.
