Function Exercise 1: Temperature Converter

It's hot out! And our silly American friends can't figure out the logical celcius system. Let's help them out by building two temperature converter functions. Hint: the formulas can be found here.

  1. Create a function called celsiusToFahrenheit that takes a celsius temperature as its argument. Convert it to fahrenheit using some fancy math and output "NN°C is NN°F" using our log function. Call the function in the console to test it out!
  2. Create a function called fahrenheitToCelsius that takes a fahrenheit temperature as its argument. Convert it to celcius using some fancy math and output "NN°F is NN°C" using our log function. Call the function in the console to test it out!
