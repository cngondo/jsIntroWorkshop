Function Exercise 2: Maximum

Time to play with some numbers! Think you can calculate if one number is larger than another using Javascript? Let's find out!

  1. Define a function max that takes two numbers as arguments and returns the largest of them. Use an if-else conditional!
  2. Define a function maxOfThree that takes three numbers as arguments and returns the largest of them. Hint: there is a method on the Math module, Math.max(), that takes 2 numerical arguments and returns the greatest of the two!
