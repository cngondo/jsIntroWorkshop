jQuery Demo

jQuery is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers. Let's try it out! Follow these instructions to see what it can do:

  1. Add two script tags at the bottom of this page's body. Connect one to index.js and the other to assets/jquery.min.js.
  2. Reload the page and see what happens!
  3. Go into index.js and use jQuery to select and change the text color on all these <li> list items. Choose whatever color you want!
  4. Set up a jQuery click listener on the triangle -> use the $(selector) syntax to target the #triangle id. Then use jQuery's .click() method and have it execute a function that targets the triangle and makes it fade away (try the jQuery .fadeOut() method.
  5. Set up a jQuery mouseenter listener on the square -> use the $(selector) syntax to target the #square id. Then use jQuery's .mouseenter() method and have it execute a function that targets the square and changes its color using the jQuery .css() method.
  6. Let's make the circle shrink on mouseenter! We'll use the same syntax that you see in sections #1 and #2. But instead of changing the CSS directly, we'll use jQuery's built-in .animate() method. Into the animate method, we pass an object that specifies how we want the animation to work. To make the circle 25px shorter on mouseenter, we can use the strings 'height' as the key and '-=25px' as the value. We should also shrink its width, to keep it a circle instead of an oval, so add another key-value pair that changes its width!
  7. The circle is shrinking out of control - it actually disappears if you mouse over it too much! Let's make it so that it grows when the mouse leaves the circle, giving it a mouseover effect of shrinking and then growing again when we leave. Use jQuery's .mouseleave() method to achieve this.
  8. Lastly, do something crazy with the rectangle! This is your own creation. Look at the jQuery documentation and make something exciting for the user. When the user clicks or touches the rectangle, make it spin or flip or change 20 colors while shrinking and flying across the screen.
