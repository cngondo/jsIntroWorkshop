Variables, Operators, Expressions

Variables, Operators and Expressions are the building blocks of the Javascript language. Let's practice using them.

  1. We'll be using console.log() a lot, so why don't we make a reusable shortcut? That's essentially what a function is - and we'll talk more about those later. In the meantime, find library.js in the Global Assets directory and let's define the following function on the first line (and use proper indentation!): var log = function(itemToLog) {
      console.log(itemToLog);
    };
     Yes, functions can be assigned as values to variables! Save that file and we should be able to use log() instead of console.log() for the rest of the course. Test it out in your console if you want!
  2. Let's start using some variables and practice some string concatenation. Open index.js and on the first line define a variable myName and assign it to a string of your name. On the next line, define a variable myLocation and assign it to a string of your current location. Last, use those variables to complete the function invocation here: log(____ + 'is learning Javascript in/at' + ______); Save that code in index.js, refresh index.html, and see what comes out on the console.
  3. The name and location are squashed against 'is' and 'at'! How can we fix that?
  4. Under that, let's start using a little math. Set a variable matatuWidth equal to 2.4 (meters). Set a variable roadWidth equal to 13 (meters). Set a variable maxMatatus equal to roadWidth divided by matatuWidth. Then use our log() function to log out the maximum number of Matatus that can possibly try to overtake each other simultaneously. If you want to get rid of any decimal places, you can put your division expression inside of the Math.round() method. Mathis a global Javascript object that holds many useful methods.
