Conditional Exercises

With conditionals, we tell javascript when or when not to execute certain pieces of code. Conditionals rely on the evaluation of expressions that return a boolean value of either true or false.

  1. We have an automated system that checks IDs at our restaurant. The system relies on a series of logical checks that makes sure we should admit the ID holder if they're over 18. Also, if they're over 60, we give them a 'Senior Discount'. An engineer built a function with most of the logic but disappeared before she could put the boolean conditions in to test the age variable. Go to index.js and put in the proper conditions!
  2. Once you're done, save index.js and refresh this page. In the console, if there are no errors, you should be able to test out the function. Type in checkID(15). Also test checkID(80) and checkID(35).
