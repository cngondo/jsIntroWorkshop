Array Exercises

Arrays are a powerful datastructure that can be used for storing all sorts of data. They are the building blocks of things like spreadsheets!

  1. In index.js, we have an array defined that contains the names of event attendees. We only want to log the people whose names begin with 'B'. Use a for loop to iterate over the list of attendees and then use a conditional to check if the first letter of their name is 'B'. Hint: you can treat strings as arrays and access their individual letters using zero-indexing! For example: 'word'[2] //'r'. Your output should look like this:
    • Beryl
    • Ben
    • Brian
    • Benoy
    • Brittany
    • Beth
    • Batman
